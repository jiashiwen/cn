# 账号列表

**1. 进入逻辑库详情页面**

可在逻辑库信息详情页面查看逻辑库的各种基本信息，包括配置、域名等

![逻辑库详情](../../../../../image/JED/keyspace_detail.png)

**2. 打开"账号管理"标签页**

点击逻辑库信息旁边的账号管理标签页，即可查看账号列表

![账号列表页](../../../../../image/JED/account_list.png)

**3. 查看明文密码**

点击查看密码，可查看明文密码

![查看明文密码](../../../../../image/JED/show_password.png)
