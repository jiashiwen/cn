# 实例管理

**1. 进入逻辑库详情页面**

可在逻辑库信息详情页面查看逻辑库的各种基本信息，包括配置、域名等

![逻辑库详情](../../../../../image/JED/keyspace_detail.png)

**2. 打开"实例管理"标签页**

点击逻辑库信息同级最右侧的实例管理标签页，即可对 tablet 进行操作

![操作实例](../../../../../image/JED/instance_management.png)

**3. 重启实例**

手动重启实例

![重启实例](../../../../../image/JED/reboot_instance.png)
