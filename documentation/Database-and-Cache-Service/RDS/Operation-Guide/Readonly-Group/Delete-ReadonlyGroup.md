# 删除只读代理
当业务不再需要某个只读实例和只读代理时，您可以删除只读代理。


## 操作步骤
1. 登录 [云数据库 RDS 控制台](https://rds-console.jdcloud.com/database)。
2. 选择需要删除只读代理的目标实例，点击目标实例的名称，进入到实例详情页。
3. 选择 **只读实例管理** 标签，进入只读代理管理页面，
4. 切换至 **只读代理** tab，选择需要删除的只读代理，点击 **删除** 按钮，删除只读代理。
